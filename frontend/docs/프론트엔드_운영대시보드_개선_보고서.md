# 프론트엔드 운영 대시보드 개선 — 작업 보고서

작업 범위: `frontend/` 폴더 한정. `/api` 계약 표면(엔드포인트·타입) 변경 없음 — 순수 프론트 시각/UX/도메인 계층. 실백엔드 연동 기준으로 검증.

이 커밋은 이번 세션의 프론트엔드 개선을 한데 담는다. 크게 **① 운영 콘솔 UX**, **② 지도 관제 마우스 회전**, **③ 기계실 관제 3D 렌더 느낌 업그레이드** 세 갈래.

---

## 1. 운영 콘솔(Ops Console) UX 개선

관련 파일: `src/ops/OpsConsole.tsx`, `src/ops/AlertDetail.tsx`, `src/ops/AgentStats.tsx`(신규)

### 1.1 클릭 없이 기본 화면 고정
- 진입/필터 변경 시 **첫 알림을 자동 선택** → 오른쪽 상세·지표 패널이 클릭 없이 바로 표시. 현재 선택 알림이 필터로 목록에서 빠지면 새 목록 첫 알림으로 자동 재선택.
- 선택 알림에 대해 **에이전트를 자동 실행**해 결과(작업지시서·지표)를 버튼 클릭 없이 기본 표시. 수동 재실행 경로는 유지. (실백엔드 연동 시 알림 선택마다 LLM이 호출되어 토큰 비용 발생 — mock에서는 무료.)

### 1.2 지표 박스 분리 및 상단 배치
- 토큰·비용 지표(모델 호출·총 토큰·입력/출력·예상 비용·단가 출처)를 **별도 박스(`AgentStats`)로 분리**해 "알림 상세·에이전트" 박스 **위**에 배치.
- 지표 박스와 작업지시서 카드 모두 **골격을 항상 고정 표시**하고, 값이 아직 없으면 자리표시자(`—`)로 채워 건물 전환 시에도 사라지지 않게 함.
- "알림 상세·에이전트" 박스는 **건물명 + 작업지시서**만 표시.
- 에이전트 실행 상태(run/result)를 `OpsConsole`로 끌어올려 지표 박스와 상세 박스가 공유.

### 1.3 표시 항목 정리
- 작업지시서 카드를 상세 패널 **맨 아래**로 이동.
- **산출물(artifacts)** 블록과 **진행 타임라인(run_started/run_completed SSE 이벤트)** 제거. 관련 훅/구독/타입 정리.

## 2. 지도 관제 — 마우스 회전/방향 전환

관련 파일: `src/map/MapView.tsx`

- MapLibre 회전 인터랙션을 명시적으로 활성화: `dragRotate.enable()`, `touchZoomRotate.enableRotation()`, `keyboard.enable()`.
- 우상단 내비게이션 컨트롤에 나침반 표시(`showCompass: true`, `visualizePitch: true`).
- 조작: **오른쪽 버튼 드래그** 또는 **Ctrl + 왼쪽 드래그** → 회전 + 기울이기 / **나침반 드래그** → 회전, **클릭** → 정북 리셋.

## 3. 기계실 관제 — 3D 렌더 느낌 업그레이드

관련 파일: `src/room/RoomSchematic.tsx`(주), `src/domain/machines.ts`, `src/theme.css`
설계 상세: [기계실_3D_업그레이드_계획서.md](./기계실_3D_업그레이드_계획서.md)

- **금속 재질**: 설비 면에 공용 스펙큘러 하이라이트(`#spec`/`#specTop`) 오버레이.
- **그림자/바닥 반사**: 설비별 접지 그림자(가우시안 블러) + base 기준 세로 미러 저투명 복제.
- **설비 기계만 애니메이션**(배관 정적):
  - 펌프 → 회전 임펠러, 제어반 → LED 점멸, 열교환기 → 히트 글로우 밴드, 탱크/밸브 → 정적.
  - **감시되는 설비만** 애니메이션(센서 미탑재=회색·정적). 긴급=빨간 펄스, 선택=시안 글로우 유지.
- **데이터 구동**: `Machine.kind` 필드로 심볼/애니메이션 결정. 31개 단지·상태·`DetailAside` 동기화 계약 유지.
- 접근성: `prefers-reduced-motion`에서 애니메이션 정지.

## 4. 검증 결과

| 항목 | 결과 |
|---|---|
| 타입체크 `npx tsc -b --noEmit` | 오류 0 |
| 운영 콘솔 진입(무클릭) | 첫 알림 자동 선택 + 지표/작업지시서 자동 표시 |
| 지표 박스 | 5개 항상 고정, 건물 전환 시 유지·값 갱신 |
| 작업지시서 카드 | 상세 패널 맨 아래, 항상 고정 표시 |
| 산출물/타임라인 | 제거됨 |
| 지도 회전 | `dragRotate.isEnabled()=true`, 나침반 존재, 회전 렌더 정상 |
| 기계실(센서 미탑재 단지) | 임펠러 2(감시 펌프)·정적 펌프 1·LED 3·긴급 pulse 1·접지그림자 7 |
| 기계실(전체 센서 단지) | 임펠러 3·LED 3·heatband 1·접지그림자 7 |
| 설비 클릭 ↔ DetailAside 동기화 | `.sel` ↔ 행 active 일치 |

> 참고: 이 환경에서 미리보기 스크린샷 도구가 뷰와 무관하게 캡처 타임아웃되어, 실제 DOM/스타일 상태(`preview_eval`)와 타입체크로 검증했다. 브라우저에서 `http://localhost:5173`로 직접 확인 가능.
