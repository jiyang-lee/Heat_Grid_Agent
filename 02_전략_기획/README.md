# 전략 기획

이 폴더는 앞으로 시뮬레이션의 방향성과 실행 계획을 쌓는 공간이다.

## 저장할 내용

- 시뮬레이션 MVP 범위
- 어떤 데이터를 어떤 시나리오로 보여줄지에 대한 계획
- 자동감지 모델, RAG, 연결/DB 산출물의 연결 방식
- 화면에서 보여줄 우선순위, 근거, 점검 포인트의 표현 방식
- 팀 산출물을 받기 전 필요한 임시 가정과 받으면 교체할 항목

## 작성 기준

- 계획은 실행 가능한 단위로 쓴다.
- 원본 데이터셋, 최종 모델, RAG/DB 산출물 중 무엇을 근거로 하는지 적는다.
- 확정된 내용과 가정은 분리한다.
- 큰 작업을 끝낸 뒤 결과 정리는 `06_보고서`에 남긴다.
